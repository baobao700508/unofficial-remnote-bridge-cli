# Overview

## Requirements[​](#requirements "Direct link to heading")

The following guides assume that you:

-   Have a basic understanding of the JavaScript or TypeScript programming languages.
-   Are somewhat familiar with the [React](https://reactjs.org/) user interface library.
-   Have a working-level understanding of [Git](https://git-scm.com/).
-   Are familiar with doing basic operations on the command line.

### Beginners[​](#beginners "Direct link to heading")

If your React skills are rusty or if you have never touched Git before, no problem! Try going through the recommended resources below first. You can finish them in less than a few hours and they will equip you the fundamental skills and knowledge you need to build RemNote plugins (and more!). After that you can move on to our [building a plugin tutorial](/in-depth-tutorial/overview) which walks you through an example project.

### Pros[​](#pros "Direct link to heading")

If you're already a React pro, you can follow our [quick start guide](/getting-started/quick_start_guide) which will get you up and running in less than 5 minutes.

### Recommended Resources[​](#recommended-resources "Direct link to heading")

-   [Tutorial: Intro to React](https://reactjs.org/tutorial/tutorial.html)
    -   This will teach you how to create React components, which are like UI lego-bricks which you can combine together into more sophisticated UIs.
-   [Git Tutorial for Beginners - Git & GitHub Fundamentals In Depth](https://www.youtube.com/watch?v=DVRQoVRzMIY)
    -   This will introduce you to Git and GitHub which we will use in the rest of the tutorials to copy the plugin template, save our work and share our code with others.

## Help and Feedback[​](#help-and-feedback "Direct link to heading")

There may be times when you aren't sure how to achieve a particular behaviour, in these situations it can be helpful to look through the code of existing plugins on GitHub.

If you get stuck or run into bugs, try asking for help in the extension-developers channel in the [RemNote Discord server](http://bit.ly/RemNoteDiscord). We have a friendly community of developers who are more than willing to help newcomers get up to speed with the plugin system.

Is there some additional functionality you need to implement your idea? We are more than happy to extend the API if possible, so please visit our [feedback platform](https://feedback.remnote.com/) to share your ideas.

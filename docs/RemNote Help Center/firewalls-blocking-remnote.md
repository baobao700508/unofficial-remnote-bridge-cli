# Firewalls Blocking RemNote

Your local RemNote app requires a connection to our servers for syncing and server features.

If you see a message "RemNote is being blocked by a firewall on your network", then something on your computer or network is interfering with RemNote's server connection. Please try:

-   Temporarily disabling any local firewalls or antivirus software
    
-   Connect to a different non-corporate or non-school network.
    
    -   We've seen firewall software like ZScalar frequently block RemNote connections.

# Working with Emojis &amp; Greek Letters

# Working with Emojis

You can use emojis in two ways in RemNote: within the text of your notes and as document icons.

## In text

You can insert emoji into RemNote notes just like any other character using your device's emoji inserter. For instance, on most phones you can tap the emoji icon, and on macOS you can press Cmd+Ctrl+Space.

If your device doesn't have a convenient emoji inserter, RemNote also includes one of its own, which you can access by typing `%%`. After typing `%%`, search for the name of the emoji you want to use, then use the arrow keys to select an appropriate item and press Enter to insert it.

## As document icons

To make it easier to spot the [document](https://help.remnote.com/en/articles/6030703-documents-and-folders) you're looking for, you can assign a document an emoji. To do so, [zoom into](https://help.remnote.com/en/articles/6030710-zooming-into-rem) the document, then choose _Add Icon_ from the _Document Style_ menu.

# Adding Greek Letters

You can easily add Greek letters to your notes – useful when studying physics, maths, chemistry and other sciences.

If you use Greek letters frequently, you may want to consider [switching your keyboard input](https://www.charlieslanguagepage.com/CWslanguagepage/How%20to%20type%20in%20Greek%20\(Windows%20and%20Mac\).pdf) for the most convenient input.

Alternatively, you can use the emoji inserter, which also supports the Greek letters. Access the emoji inserter by typing `%%` and searching for the name of the letter you want to use.

[![](https://downloads.intercomcdn.com/i/o/1111644425/b03715fb6f64d603e031e247/Screenshot+2024-07-12+at+19_00_11.png?expires=1772827200&signature=26b2abdb89bfb65e518c91d56c106cf5cd92d22d9faf8c327b1ac49f9d3e49c8&req=dSEmF896mYVdXPMW1HO4zVmmk7HIqNr8jPfsnFef8Ekj%2FHuO6Xg5Kf4ka4hQ%0AYoKQpDhY30%2Fd0De1OsE%3D%0A)](https://downloads.intercomcdn.com/i/o/1111644425/b03715fb6f64d603e031e247/Screenshot+2024-07-12+at+19_00_11.png?expires=1772827200&signature=26b2abdb89bfb65e518c91d56c106cf5cd92d22d9faf8c327b1ac49f9d3e49c8&req=dSEmF896mYVdXPMW1HO4zVmmk7HIqNr8jPfsnFef8Ekj%2FHuO6Xg5Kf4ka4hQ%0AYoKQpDhY30%2Fd0De1OsE%3D%0A)

If you want to use Greek letters in mathematical or chemical equations, you can instead use LaTeX: type either `$$` or `/latex`, then write/paste the mathematical/chemical text into the LaTeX Equation Field pop-up window using LaTeX `\macros`.

[![](https://downloads.intercomcdn.com/i/o/1111650380/7e8541693ddcffac6f885112/Screenshot+2024-07-12+at+19_21_26.png?expires=1772827200&signature=221677270e9e588a4f993ac12c42eddabefb3d855e1ffc5f0cdf04e1ae8be2af&req=dSEmF897nYJXWfMW1HO4zeAGZ5nZ%2FKZO1NrlD5MzugoL9YPtePdwF%2FhLhc%2Bo%0AJc3wWDEM%2Bl84e%2BzMP2E%3D%0A)](https://downloads.intercomcdn.com/i/o/1111650380/7e8541693ddcffac6f885112/Screenshot+2024-07-12+at+19_21_26.png?expires=1772827200&signature=221677270e9e588a4f993ac12c42eddabefb3d855e1ffc5f0cdf04e1ae8be2af&req=dSEmF897nYJXWfMW1HO4zeAGZ5nZ%2FKZO1NrlD5MzugoL9YPtePdwF%2FhLhc%2Bo%0AJc3wWDEM%2Bl84e%2BzMP2E%3D%0A)

A list of LaTeX macros that allow you to add Greek characters can be found in the documentation [here](https://katex.org/docs/supported#letters-and-unicode), and more information about how LaTeX works can be found in this article: [Writing equations with LaTeX](https://help.remnote.com/en/articles/6565191-writing-equations-with-latex).

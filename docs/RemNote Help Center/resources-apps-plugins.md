# Resources, Apps, &amp; Plugins

Take a look at some official and fantastic community-made tutorials, browser extensions, plugins, and themes to add even more functionality to RemNote.

# Apps

-   [RemNote Desktop Apps for Mac, Windows, and Linux](https://www.remnote.com/download)
    
-   [RemNote Mobile Apps for Android and iOS](https://www.remnote.com/mobile)
    

# Tutorials

-   **[Official YouTube tutorials](https://www.youtube.com/watch?v=E6xQtbEnXVc&list=PLBwTyifkXByjkj_H8OFTCPAtoc58Txzt0):** Learn RemNote by following along with visual lessons in the form of YouTube videos.
    

# Chrome Extensions

-   **[RemNote Clipper](https://chromewebstore.google.com/detail/remnote-clipper/lpnnhhkdbmobjiheokoonjlmaffljcng?hl=en)**: Save web pages and highlights to your notes from any web page. (If you are planning to thoroughly read and digest a page, also consider importing it into RemNote and reading it with the [web reader](https://intercom.help/remnote/en/articles/7910548-learning-from-web-pages-in-the-remnote-reader)!)

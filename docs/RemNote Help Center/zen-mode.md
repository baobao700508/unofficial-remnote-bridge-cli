# Zen Mode

If you find interface elements distracting while writing, you can enable Zen Mode to automatically hide the [sidebar](https://help.remnote.com/en/articles/6030849-customizing-the-document-sidebar), top toolbar, and tabs whenever you start typing. When you stop typing, they reappear so you can navigate and access tools as usual.

To toggle Zen Mode on or off, go to _Settings > Editor_ and check or uncheck the Zen Mode option under the _Animations_ section.

[![](https://downloads.intercomcdn.com/i/o/akxf7g7x/2086631769/6c8ae67e5b8fe96d1f380a576edc/CleanShot+2026-02-20+at+18_15_21%402x.png?expires=1772827200&signature=e989100477ed123b64a9c0b25c83e20cfc36137a3b00c7050139637f76aacf49&req=diAvEM99nIZZUPMW1HO4zSWF97AxyCWeQcBkPw3SDdhAf59PEUvSnDLYKt1J%0AHS%2BzLVImfTeLZUuVWG0%3D%0A)](https://downloads.intercomcdn.com/i/o/akxf7g7x/2086631769/6c8ae67e5b8fe96d1f380a576edc/CleanShot+2026-02-20+at+18_15_21%402x.png?expires=1772827200&signature=e989100477ed123b64a9c0b25c83e20cfc36137a3b00c7050139637f76aacf49&req=diAvEM99nIZZUPMW1HO4zSWF97AxyCWeQcBkPw3SDdhAf59PEUvSnDLYKt1J%0AHS%2BzLVImfTeLZUuVWG0%3D%0A)

You can also toggle Zen Mode on the fly using the [Omnibar](https://help.remnote.com/en/articles/7852647-using-the-omnibar) — press Ctrl+K (Cmd+K on a Mac) and search for _Zen Mode_.

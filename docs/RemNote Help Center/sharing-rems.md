# Sharing Rems

RemNote makes it easy to share any subset of your notes with other people. This can be useful for publishing your notes (explained below) or for creating flashcards [collaboratively](https://help.remnote.com/en/articles/7322908-how-can-i-create-flashcards-with-someone-else) in a shared Knowledge Base.

We also have a guide on [how to create a great shared document](https://help.remnote.com/en/articles/7727218-how-to-publish-a-great-remnote-document).

# Sharing and Resharing

From any document, click Share in the upper-right corner. You can choose to list your notes publicly, or keep them unlisted so that only people you give the link to can see them.

If your document is published publicly, other users will be able to see it on [your profile page](https://help.remnote.com/en/articles/8119315-groups-and-profiles#h_d788b63840).

Once the document is shared, click _Copy Link_ and share the link wherever you like.

Sharing creates a snapshot of your document in its current state. If you share the document again, the shared version will be brought up to date (keeping the same link so others who have the link see the changes). This allows you to edit your documents in private and only share the updates when you're ready for others to see them.

# Unsharing

If you decide you no longer want to share a document, open the document and select _Unshare Document_ from the … menu in the upper-right corner.

[![](https://downloads.intercomcdn.com/i/o/akxf7g7x/1901655025/a63aa1c8f2775f57e55f659ebc10/CleanShot+2025-12-24+at+17_42_31%402x.png?expires=1772827200&signature=ecb1e516f5572a10bb0e6e67f42f0fe8c50c6630be7269e0977500920f13a876&req=dSknF897mIFdXPMW1HO4zU7W%2Bm%2FL2oQPuM0Mwa73xFg0G%2F6OJt8FdzuWkNqz%0ArOgAg7cTB3%2BtuVgzuCs%3D%0A)](https://downloads.intercomcdn.com/i/o/akxf7g7x/1901655025/a63aa1c8f2775f57e55f659ebc10/CleanShot+2025-12-24+at+17_42_31%402x.png?expires=1772827200&signature=ecb1e516f5572a10bb0e6e67f42f0fe8c50c6630be7269e0977500920f13a876&req=dSknF897mIFdXPMW1HO4zU7W%2Bm%2FL2oQPuM0Mwa73xFg0G%2F6OJt8FdzuWkNqz%0ArOgAg7cTB3%2BtuVgzuCs%3D%0A)

All existing links to the document will stop working, and it will no longer be listed on your profile page if it was publicly listed there.

Be aware that if you later choose to re-share the Rem, a new sharing link will be generated; the old link will not work again.

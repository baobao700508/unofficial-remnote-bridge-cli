# Configuring Advanced Tables

The functionality and appearance of RemNote [tables](https://help.remnote.com/en/articles/8118336-tables) can be customized in a number of ways.

# Sorting

You can change the order of rows in a table by clicking the _Sort_ button, then choosing a column to sort on (in alphabetical order).

[![](https://downloads.intercomcdn.com/i/o/794000129/92c0df9c922b40a5577698a4/2023-07-26-16-44-08%402x.png?expires=1772827200&signature=3795cbe6f09ae88c54a953f5772fc043cb5c4991f96d3cbed5d14b1c3f792f12&req=cykjFsl%2BnINWFb4f3HP0gF6Gy72NQ4y6O6huxRGghGh93HdwfRES7s5qnPk5%0A97QD%2BG41kHVHlXJLCA%3D%3D%0A)](https://downloads.intercomcdn.com/i/o/794000129/92c0df9c922b40a5577698a4/2023-07-26-16-44-08%402x.png?expires=1772827200&signature=3795cbe6f09ae88c54a953f5772fc043cb5c4991f96d3cbed5d14b1c3f792f12&req=cykjFsl%2BnINWFb4f3HP0gF6Gy72NQ4y6O6huxRGghGh93HdwfRES7s5qnPk5%0A97QD%2BG41kHVHlXJLCA%3D%3D%0A)

Once you’ve picked an initial sort column, clicking the _Sort_ button again will give you additional options:

-   You can change a column’s sort from _Ascending_ to _Descending_ to reverse the sort order (showing Z higher in the list than A):
    
    [![](https://downloads.intercomcdn.com/i/o/794000450/926d62503e78fa1aedc2f71d/2023-07-26-16-45-25%402x.png?expires=1772827200&signature=3cf667d15e3a450370adedca6096e1a66551f716352d3914946e83f5a82a0e5a&req=cykjFsl%2BmYRfFb4f3HP0gHYg1UP4A6cs5j8U%2FLWLKq1qxHzoOm1IJ9P7u3LF%0A2NA%3D%0A)](https://downloads.intercomcdn.com/i/o/794000450/926d62503e78fa1aedc2f71d/2023-07-26-16-45-25%402x.png?expires=1772827200&signature=3cf667d15e3a450370adedca6096e1a66551f716352d3914946e83f5a82a0e5a&req=cykjFsl%2BmYRfFb4f3HP0gHYg1UP4A6cs5j8U%2FLWLKq1qxHzoOm1IJ9P7u3LF%0A2NA%3D%0A)
    
-   You can add additional sort columns. If you have more than one sort column selected, all rows will first be put in order by column 1; then if any rows had the same value in column 1, those rows will be put in order by column 2, and so on for additional sort columns.

# Filtering

You can limit a table to show only rows that match some search criteria. Start by clicking the _Filter_ button and selecting a property to filter by. Then choose a value to filter on. The method of selecting a value will depend on the [type of the property](https://help.remnote.com/en/articles/8126585-properties-tables-alpha#h_359c6276fb) – for instance, for single-select or multi-select properties, you’ll be asked to select from a list, while for text properties, you’ll be asked to type in a value.

# Searching

In addition to the restrictions imposed by filters, you can instantly filter a table to show only rows that match a particular string of text in some cell using the search box above the table.

# Hiding Properties

When you create a table, a column is added for each property defined on the table’s tag. If you don’t want to see a particular property as a column in this table, simply choose _Hide Property_ from the column’s drop-down to remove it.

[![](https://remnote.intercom-attachments-1.com/i/o/794022264/00b16f80f20bb6a688a58368/thumb-4cd586699a3fb01a279c54871904dd1e.gif?expires=1772827200&signature=869413c618d9d1006d8713ad2416cb8631b17463dc3151780c3847933a42cae8&req=cykjFst8n4dbFb4f3HP0gOls9%2FtHnDi5X507DD7x4MKYMoIBmXshbw1t2N1H%0A1sybI8EA5SXO2cpkeQ%3D%3D%0A)](https://remnote.intercom-attachments-1.com/i/o/794022264/00b16f80f20bb6a688a58368/thumb-4cd586699a3fb01a279c54871904dd1e.gif?expires=1772827200&signature=869413c618d9d1006d8713ad2416cb8631b17463dc3151780c3847933a42cae8&req=cykjFst8n4dbFb4f3HP0gOls9%2FtHnDi5X507DD7x4MKYMoIBmXshbw1t2N1H%0A1sybI8EA5SXO2cpkeQ%3D%3D%0A)

This doesn’t delete the data – you can still view and edit it by zooming in to the row (hover over the _Name_ column for that row and click the _Open_ button).

# Cell List Height

Properties can contain bulleted lists of Rems, which cause the table row to stretch to multiple lines. You can control how much space is used within each table cell with the _Cell List Height_ option on the Table menu (upper-right corner of the table). _Full_ uses the same spacing as in the editor outside of a table cell and shows all bullets, _Comfortable_ makes the space between them slightly smaller and shows only the first three bullets, and _Compact_ squashes them onto one line. Hover over the _Cell List Height_ option to see a demo of these options.

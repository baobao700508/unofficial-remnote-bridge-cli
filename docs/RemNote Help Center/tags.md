# Tags

Tags allow you to describe **what kind of thing a Rem is**, in a way that cuts across all [documents](https://help.remnote.com/en/articles/6030703-documents-and-folders) and [hierarchies](https://help.remnote.com/en/articles/8196578-outlines-and-terminology) in your Knowledge Base. For instance, you might have an “Author” tag you apply to Rems representing authors you like, or an “Urgent” tag which you place on [todo items](https://help.remnote.com/en/articles/6752161-managing-tasks-with-todos) that you need to complete very soon. You can attach a tag to a Rem by putting your cursor within the Rem you want to tag, then typing `##` and the tag's name; or by copying the tag, then pasting it over the Rem you want to tag.

Tags, References, and Portals are all ways of relating Rems to each other. If you're not sure which one to choose, check out [What's the difference between References, Tags, and Portals?](https://help.remnote.com/en/articles/6634227-what-s-the-difference-between-references-tags-and-portals)

Tags are closely integrated with [properties](https://help.remnote.com/en/articles/8126585-properties-tables-alpha), [templates](https://help.remnote.com/en/articles/8117687-templates-tables-alpha), and [advanced tables](https://help.remnote.com/en/articles/8118336-tables). For a quick overview of how these all connect, see [Tags, Properties, Tables, and Templates in Five Minutes](https://help.remnote.com/en/articles/8070716-tags-properties-tables-and-templates-in-five-minutes-tables-alpha).

# Tags are Rems

Because [everything is a Rem](https://help.remnote.com/en/articles/8017859-rem), a tag is just a plain old Rem that has some additional metadata attached to it. This means you can use any Rem, located anywhere in your knowledge base, to tag any other Rem! For instance, you might want to take notes on what a _Chemical Element_ is (the history of the idea, how elements are organized in the periodic table, and so on), making _Chemical Element_ into a heading or [Concept](https://help.remnote.com/en/articles/6026154-structuring-knowledge-with-the-concept-descriptor-framework) and adding notes beneath it. But you might also want to tag each element you take notes on with _Chemical Element_. Since _Chemical Element_ is a Rem, you can do both, and the _Chemical Element_ Rem will keep track of both those notes and a list of all the other elements you tagged.

# Tagging a Rem

To apply a tag to a Rem, type # twice in your editor. Then select an existing tag or type a new name and choose _Create Rem_ (or press Ctrl+Enter, or Cmd+Return on a Mac).

Since tags are just Rems, you can theoretically use any Rem in your knowledge base as a tag. To make it easier to find the Rems that you’re likely to want to tag something with, the `##` tag search menu starts by only showing Rems that:

-   have previously been used as a tag;
    
-   have properties or templates assigned; or
    
-   have _exactly_ the same name as what you typed.
    

If you want to start using some other Rem as a tag, just choose the _Show Non-Tags_ option in the list to expand the search to all Rems:

Editing a tag

Since, again, tags are just Rems, you can access them in the main editor, rename them by changing their text, move them into hierarchies, etc.

At the bottom of a tag's page, all the Rems it's tagging will be listed. Here, we open up the _Urgent_ tag by clicking on it, look at the list of Rems it's tagging, and change its name to _Super Urgent_.

# Filtering by tag

Sometimes it's helpful to show only the Rems in some document that have a particular tag, hiding everything else. For instance, you can hide all to-do items except those that are urgent. There are two ways to do this:

-   Right-click on some instance of that tag and select _Tag Options > Filter Document by This Tag_:
    
    [![](https://downloads.intercomcdn.com/i/o/akxf7g7x/1348627633/87e549ae68023cdb29c8e1add16b/image.png?expires=1772827200&signature=942c2ea8f9382403f990fd8c6e9f8100b4109c6c08ffde7e392d45f88a8904b5&req=dSMjHs98modcWvMW1HO4zblKKPZEXwEZlfdIxMarJT0IesvTqK3i2G%2BklNT2%0AbjOH%0A)](https://downloads.intercomcdn.com/i/o/akxf7g7x/1348627633/87e549ae68023cdb29c8e1add16b/image.png?expires=1772827200&signature=942c2ea8f9382403f990fd8c6e9f8100b4109c6c08ffde7e392d45f88a8904b5&req=dSMjHs98modcWvMW1HO4zblKKPZEXwEZlfdIxMarJT0IesvTqK3i2G%2BklNT2%0AbjOH%0A)
    
-   Press Ctrl+F (Cmd+F on a Mac), select _Filter_ from the search mode drop-down, and select the tag you want to filter on.

# Software Updates

To keep your RemNote experience fresh, we release updates on a regular basis, usually multiple times a week. All RemNote apps receive new updates automatically.

If you ever need to check what version you’re currently using, you can find it in _Settings > Troubleshooting:_

## Update schedule

On the **web app**, you’ll receive RemNote updates every time the page reloads. To check for updates and get the latest version sooner, simply refresh the page (on most browsers, this is Cmd+R on Mac, and Ctrl+R or F5 on Windows and Linux).

On the **mobile app**, updates are managed through your device’s app store (Google Play on Android devices, Apple App Store on iOS devices). If you have questions about how app store updates work, refer to the documentation provided with your device.

On the **desktop app**, you’ll be prompted in the lower-left corner to reload the app when updates are available. Simply click _Update and Reload_ to complete the update. If you don’t choose this option, next time you close and reopen RemNote, you’ll get the updated version.

Desktop app updates may take a few hours to show up after an update is released. To force RemNote to check for updates immediately, use the _Check for Updates_ button in Settings > Desktop App:

If the desktop app ever gets stuck while updating, you can fix it by manually downloading and installing the latest version from [https://remnote.com/download](https://remnote.com/download).

## Beta version

By default, you’ll only receive updates to the main release version. If you want to try out the beta version for early access to new changes, check out the article on [prereleases](https://help.remnote.com/en/articles/7868812-prereleases-beta-version).

# How do I switch from numbers to bullets?

RemNote lets you toggle any Rem between **numbered lists** and **bulleted lists** at any time. This is useful when you want to organize steps in a sequence or switch back to a more flexible outline format.

# Switching Between Numbers and Bullets

There are several ways to toggle a Rem between numbered and bulleted display:

## Type a number to start a numbered list

Type a number followed by a period (e.g., `1.`) at the very start of a Rem to convert it to a numbered list item. To switch it back to a bullet, use the slash command or omnibar (see below).

## Use the toolbar

1.  Select rems you want to change
    
2.  In the toolbar, open the _Turn into_ menu > _Numbered list_
    

[![](https://downloads.intercomcdn.com/i/o/akxf7g7x/2122373571/b48d614e47b1890d885260b199cf/CleanShot+2026-03-03+at+12_11_32%402x.png?expires=1772827200&signature=55f34490afc47f6c49721cf3a2b80a421c1d2b2b4b326e1e0484045f1f4bd21b&req=diElFMp5noRYWPMW1HO4zXvQt7SduN5q%2FmuzAe7k6cxFa173li1kYON2dsl3%0AElX%2BsVMLD8KX%2FcjPxEY%3D%0A)](https://downloads.intercomcdn.com/i/o/akxf7g7x/2122373571/b48d614e47b1890d885260b199cf/CleanShot+2026-03-03+at+12_11_32%402x.png?expires=1772827200&signature=55f34490afc47f6c49721cf3a2b80a421c1d2b2b4b326e1e0484045f1f4bd21b&req=diElFMp5noRYWPMW1HO4zXvQt7SduN5q%2FmuzAe7k6cxFa173li1kYON2dsl3%0AElX%2BsVMLD8KX%2FcjPxEY%3D%0A)

## Use the slash command

1.  Place your cursor in the Rem you want to change (or select multiple Rems).
    
2.  Type `/list item` and select it from the menu.
    

You can also press `Ctrl+Alt+L` (`Cmd+Opt+L` on Mac) as a [keyboard shortcut](https://help.remnote.com/en/articles/7893440-keyboard-shortcuts) instead of opening the slash menu.

## Use the omnibar

1.  Place your cursor in the Rem you want to change (or select multiple Rems).
    
2.  Open the [omnibar](https://help.remnote.com/en/articles/7852647-using-the-omnibar) with `Ctrl+K` (`Cmd+K` on Mac).
    
3.  Type `list item` and select the action.

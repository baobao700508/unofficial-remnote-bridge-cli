# Read-Only Mode

When reading a lot of notes at a time on a smartphone or other mobile device, it can sometimes be difficult to scroll through them without accidentally selecting a Rem and starting to edit it, thus opening the keyboard and using up most of your screen space. If you're only reading and not editing, you can temporarily enable _read-only mode_ to avoid this annoyance.

Find Read-Only Mode on the … menu in the upper-right corner of the screen. After selecting this option, tapping on text will no longer enter edit mode, but you'll still be able to scroll and expand or collapse existing Rems. To exit this mode and resume editing your documents, simply press the button again.

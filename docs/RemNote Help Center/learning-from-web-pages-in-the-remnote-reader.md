# Learning from Web Pages in the RemNote Reader

For background on reading and annotation in RemNote, review [Learning from PDFs and Files with the RemNote Reader](https://help.remnote.com/en/articles/6690975-learning-from-pdfs-and-files-with-the-remnote-reader). This article will explain only what differs when annotating a web page.

The Web Reader is a RemNote Pro feature.

# Getting started

You can import a web page to RemNote by pasting a link into the [upload popup](https://help.remnote.com/en/articles/6690972-uploading-pdfs-to-remnote), just like you would do with the URL of a file. For instance, here's a Wikipedia article:

# Highlighting

Highlighting largely works the same way in the web reader as the PDF reader: you select the content to highlight by dragging over it and then releasing the mouse button.

[![](https://downloads.intercomcdn.com/i/o/akxf7g7x/1261418454/d52b113034a72977db10399552bd/image.png?expires=1772827200&signature=8bedbeb7ea6b15d351c7518a95f067beafce9e3d8e026cabf7898fe5394c8b32&req=dSIhF81%2FlYVaXfMW1HO4zbeowh52xN8RWWLAVcAUIKiuFaVJuTSfQ68YqkrM%0A%2Bw5zZEIa%2B%2BrfZa4X%2F8w%3D%0A)](https://downloads.intercomcdn.com/i/o/akxf7g7x/1261418454/d52b113034a72977db10399552bd/image.png?expires=1772827200&signature=8bedbeb7ea6b15d351c7518a95f067beafce9e3d8e026cabf7898fe5394c8b32&req=dSIhF81%2FlYVaXfMW1HO4zbeowh52xN8RWWLAVcAUIKiuFaVJuTSfQ68YqkrM%0A%2Bw5zZEIa%2B%2BrfZa4X%2F8w%3D%0A)

Area (“box”) highlights aren’t supported in the web reader. However, most types of content, including images, can be directly captured by text highlights.

# Right Sidebar

All of the right sidebar tabs like _Summary_, _AI Tutor_, and _Editor_ work the same way in the web reader as the PDF reader.

# Limitations

Some options present in the PDF reader aren’t currently available in the web reader. For instance, you can’t zoom the web page or change the theme, and Key Points are not available. Many of these options will likely be added in the future.

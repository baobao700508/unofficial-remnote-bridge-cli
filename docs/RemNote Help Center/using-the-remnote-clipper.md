# Using the RemNote Clipper

The [RemNote Clipper Chrome extension](https://chromewebstore.google.com/detail/remnote-clipper/lpnnhhkdbmobjiheokoonjlmaffljcng) allows you to save a web page to your knowledge base and make quick notes on it. You can also show the RemNote editor and the web page side by side so you can copy text or take notes on the page.

**Tip:** If you're planning to do extensive annotation of a page and you have a RemNote Pro subscription, the [web reader](https://help.remnote.com/en/articles/7910548-annotating-web-pages-in-remnote) may be a better fit than the Clipper.

# **Installation**

The [RemNote Clipper](https://chromewebstore.google.com/detail/remnote-clipper/lpnnhhkdbmobjiheokoonjlmaffljcng) is available on Google Chrome and other Chrome-based browsers via the Chrome Web Store. To install it, click on that link or search for _RemNote Clipper_ in the store, then click the _Add to Chrome_ button and follow any prompts.

# **Taking Notes**

When you're viewing a web page you would like to save or take notes on, click the RemNote plugin icon.

[![](https://downloads.intercomcdn.com/i/o/715681370/6425064ba64b02db115b509a/image.png?expires=1772827200&signature=b3e9f7332e94726006417f81ed53286aa09389f34b97d35fee2aa747a2b5dbcb&req=cyEiEMF%2FnoZfFb4f3HP0gJ0aBZa9XeYN3bBxdzXLRECBTP%2B3fyx0ERe4jxgj%0Awj7LbbtlJSATMpY4Nw%3D%3D%0A)](https://downloads.intercomcdn.com/i/o/715681370/6425064ba64b02db115b509a/image.png?expires=1772827200&signature=b3e9f7332e94726006417f81ed53286aa09389f34b97d35fee2aa747a2b5dbcb&req=cyEiEMF%2FnoZfFb4f3HP0gJ0aBZa9XeYN3bBxdzXLRECBTP%2B3fyx0ERe4jxgj%0Awj7LbbtlJSATMpY4Nw%3D%3D%0A)

To save the web page into a new document in your knowledge base under the _Links_ power-up, type a quick note about why you're saving the page or what you want to remember about it if you wish, then click _Save note_ or press Ctrl+Enter (Cmd+Return on a Mac).

[![](https://downloads.intercomcdn.com/i/o/1144144941/54ff09b155b3f06c42666744/image.png?expires=1772827200&signature=8031af824b290f251bd9fd456e89341e569e7a5d162a364dbe22d26c67c8a6ba&req=dSEjEsh6mYhbWPMW1HO4zUx9bQGehXVsRWcXZmGnnGda0hQvd%2BOOHDExlQv9%0AnXEh%2BjihKuMyTDSVtcA%3D%0A)](https://downloads.intercomcdn.com/i/o/1144144941/54ff09b155b3f06c42666744/image.png?expires=1772827200&signature=8031af824b290f251bd9fd456e89341e569e7a5d162a364dbe22d26c67c8a6ba&req=dSEjEsh6mYhbWPMW1HO4zUx9bQGehXVsRWcXZmGnnGda0hQvd%2BOOHDExlQv9%0AnXEh%2BjihKuMyTDSVtcA%3D%0A)

You can open this popup and choose “Save note” repeatedly to add multiple notes on the page, if you like. Each note that you type into the box will become a new Rem in the page's document.

To work with a larger number of notes, click _Edit in Side-panel_ or press Alt+Enter. The document will open in a RemNote editor window to the right:

[![](https://downloads.intercomcdn.com/i/o/1144146123/dc4db8842f01d082a17e3388/image.png?expires=1772827200&signature=6883c2500c8be9ff25d9d1d5f7a3e2c5c77b342e152d355131cfc90cbf8cf9b7&req=dSEjEsh6m4BdWvMW1HO4zZzFr6MqvEHH6BuBf%2F4OBzxtKZXduiibXnq3AWfD%0AubJc5JLfV%2BnTkW0mOjM%3D%0A)](https://downloads.intercomcdn.com/i/o/1144146123/dc4db8842f01d082a17e3388/image.png?expires=1772827200&signature=6883c2500c8be9ff25d9d1d5f7a3e2c5c77b342e152d355131cfc90cbf8cf9b7&req=dSEjEsh6m4BdWvMW1HO4zZzFr6MqvEHH6BuBf%2F4OBzxtKZXduiibXnq3AWfD%0AubJc5JLfV%2BnTkW0mOjM%3D%0A)

The editor window works exactly the same way as a full RemNote window – you can use the /-menu, search, navigate to other documents, and even open multiple panes here, as needed.

Once you have the split screen open, selecting text in the web page will offer an _Add Note_ option to copy it directly to a new Rem in your notes.

# Dark Mode

If you're taking notes in a dimly lit environment or prefer dark themes in general, you can display RemNote's interface in dark mode by selecting the appropriate option in _Settings > Interface:_

[![](https://downloads.intercomcdn.com/i/o/akxf7g7x/2086517372/85845f6e434eaba33a819947841b/CleanShot+2026-02-20+at+17_39_59%402x.png?expires=1772827200&signature=c037573b7e01a8fa4dfc87cef9d2eead800be1f677ff8bfde6b1de4629b41ed4&req=diAvEMx%2FmoJYW%2FMW1HO4zUTkOP4DgM01FQG2OB7%2BKtFokbigHyCZRrbqB0%2FM%0AIfzQv40zv%2FzDZ83NyhU%3D%0A)](https://downloads.intercomcdn.com/i/o/akxf7g7x/2086517372/85845f6e434eaba33a819947841b/CleanShot+2026-02-20+at+17_39_59%402x.png?expires=1772827200&signature=c037573b7e01a8fa4dfc87cef9d2eead800be1f677ff8bfde6b1de4629b41ed4&req=diAvEMx%2FmoJYW%2FMW1HO4zUTkOP4DgM01FQG2OB7%2BKtFokbigHyCZRrbqB0%2FM%0AIfzQv40zv%2FzDZ83NyhU%3D%0A)

The _System_ option will follow your operating system’s settings, if it has them (this can be helpful, for instance, if you have your computer configured to switch to dark mode after sunset).

Unlike most settings, the current value of the dark mode setting is maintained individually for each device you use RemNote on, so you can, for example, use dark mode on your laptop and light mode on your phone, without having to toggle it back and forth all the time.

You can also toggle dark mode on the fly using the _Toggle Dark Mode_ [omnibar](https://help.remnote.com/en/articles/7852647-using-the-omnibar) action:

# Dark mode in the RemNote Reader

By default, the [RemNote Reader](https://help.remnote.com/en/articles/6690975-learning-from-pdfs-and-files-with-the-remnote-reader) will use the same dark mode setting when displaying PDFs and other documents as the rest of RemNote's interface. Occasionally this will make the document hard to read, and you might prefer to use a different color scheme for documents anyway. You can override the color scheme for documents with the _Appearance_ option on the Reader menu:

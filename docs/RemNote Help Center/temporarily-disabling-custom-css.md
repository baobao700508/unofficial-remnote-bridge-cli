# Temporarily Disabling Custom CSS

If you add `?disableCustomCSS` at the end of any URL (for example, [https://remnote.com/notes?disableCustomCSS](https://remnote.com/notes?disableCustomCSS)), that will turn off loading of [custom CSS](https://help.remnote.com/en/articles/8061994-custom-css) for that session. This can come in handy if you are editing custom CSS and accidentally break the formatting so badly that you can't get make your way back to the custom CSS page to change it again.  
​  
See more documentation at [the bottom of the custom CSS docs page](https://plugins.remnote.com/custom-css#custom-css-escape-hatch).

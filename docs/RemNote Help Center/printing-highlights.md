# Printing Highlights

By default, RemNote doesn't include background highlight colors in printouts, because they make the document hard to read on many printers (especially lower-quality black-and-white printers).

If this isn't an issue with your printer and you'd prefer to include the highlights, you can enable _Background Graphics_ in the print options to print the highlight color too. Please note that this only works on Google Chrome.

# What&#x27;s the difference between References, Tags, and Portals?

References, Tags, and Portals are all ways of using the same content in multiple places and defining how things relate to each other. However, they have distinct functions.

# Summary

Suppose you have two Rems, creatively called **A** and **B**, and you're editing A and want to refer to B.

**Reference**

**Tag**

**Portal**

**In a few words**

A link to another Rem

A note that **A** goes in category **B**

A window showing another Rem

**Implies that**

**B** is related to **A**

**A** is a type of **B**

**B** is used in the context of **A**

**How to create it**

Type `[[` or `@`, then **B**

Type `##`, then **B**

Type `((`, then **B**

**Everything it does**

-   Shows a link to **B** within the text of **A**.
    
-   Creates a backlink from **B** to **A**.
    

-   Shows the name of **B** next to the Rem **A**.
    
-   Creates a backlink from **B** to **A**.
    
-   Gives access to the [properties](https://help.remnote.com/en/articles/8126585-properties-tables-alpha) and [templates](https://help.remnote.com/en/articles/8117687-templates-tables-alpha) defined by **B**, if they exist, to **A**.
    

-   Shows the full content of **B** right within **A**, updating all copies in real time if you make changes.
    
-   Creates a backlink from **B** to **A**.

# References

A Reference is simply **a link to another Rem**. The name of the Rem you reference appears in blue, and clicking on it takes you to the referenced Rem.

Use References when you want to mention some other related concept or piece of information, but don't think you'll need to immediately revisit its definition when rereading the notes later. If you think you will need the definition, you might prefer a Portal instead for greater convenience.

There's one additional trick to References: In RemNote, all links are bidirectional. That is, if you create a reference from _Keyboard_ to _Computer_, you can not only see this link when looking at the _Keyboard_ Rem, but also when looking at the _Computer_ Rem. This backwards link (often called a _backlink_) appears under a _References_ section which you can access by clicking the numbered bubble to the right of the _Computer_ Rem.

[![](https://downloads.intercomcdn.com/i/o/596245714/a8a5dcc278123f846ea06c0b/image.png?expires=1772827200&signature=75bf35e7a82652d742ad50827cd8b16ba794337b17347dcae2fd4472ca790aea&req=cSkhFM17moBbFb4f3HP0gLymIBU9SIfsE6oK0HmlUeBmTWy%2FmLHterHNbZDV%0AHbwe5po%2FtFDJShNhaQ%3D%3D%0A)](https://downloads.intercomcdn.com/i/o/596245714/a8a5dcc278123f846ea06c0b/image.png?expires=1772827200&signature=75bf35e7a82652d742ad50827cd8b16ba794337b17347dcae2fd4472ca790aea&req=cSkhFM17moBbFb4f3HP0gLymIBU9SIfsE6oK0HmlUeBmTWy%2FmLHterHNbZDV%0AHbwe5po%2FtFDJShNhaQ%3D%3D%0A)

The simplest way to create a Reference is to press `[[` while typing; this will bring up an inline search box which you can use to find the Rem you want to reference.

[Learn more about References](https://help.remnote.com/en/articles/6030714-rem-references).

# Tags

Tagging one Rem with another means that the tagged Rem is **a type of** **the tagging Rem**. To make that more concrete, a _Keyboard_ and a _Mouse_ are both types of _Hardware_, so you might apply the _Hardware_ tag to the _Keyboard_ and _Mouse_ Rems.

As with References, tags are bidirectional; you can see all of the places a tag is used by clicking the numbered bubble on the right side of that tag's Rem.

[![](https://downloads.intercomcdn.com/i/o/596250544/b58a4941e9c0c9ca6de27c69/image.png?expires=1772827200&signature=644e2294084dff79ff9ace2a2603dbc934b5aa2861de92c0e8c6229ac81f830c&req=cSkhFMx%2BmIVbFb4f3HP0gAghHhMBHru12XVoDgavBE2cTZUi8vk3qyRwihMx%0AH0S5TXx8RNbZIBriaw%3D%3D%0A)](https://downloads.intercomcdn.com/i/o/596250544/b58a4941e9c0c9ca6de27c69/image.png?expires=1772827200&signature=644e2294084dff79ff9ace2a2603dbc934b5aa2861de92c0e8c6229ac81f830c&req=cSkhFMx%2BmIVbFb4f3HP0gAghHhMBHru12XVoDgavBE2cTZUi8vk3qyRwihMx%0AH0S5TXx8RNbZIBriaw%3D%3D%0A)

The simplest way to add a tag to a Rem is to press `##` while typing; this will bring up an inline search box which you can use to find the Rem you want to tag the current Rem with.

[Learn more about tags](https://help.remnote.com/en/articles/6030770-tags).

# Portals

A Portal **displays a particular Rem**, and optionally some or all of its [descendants](https://help.remnote.com/en/articles/8196578-outlines-and-terminology#h_9808744ba6), at an arbitrary place **within some other notes**. You can think of a portal like a window cut in the page with the source document placed behind it: you can see all the material at the other location right within the current document, but it's actually located in a different document, and if you update the information in one place, it will be updated in the original location as well as all its portals.

Use Portals when you want to keep information that is relevant in many contexts in one centralized place, but include it directly within your notes on particular sources or more specific topics for easy review.

You can tell that content is part of a portal when there is a blue bar at the left side of the content. When you click inside the portal, the bar extends into a full border around the entire portal.

Again, portals are bidirectional; the numbered bubble will show any places where the current Rem has been included in a portal (unless the portal is within the current document).

[![](https://downloads.intercomcdn.com/i/o/596255773/72d4fe3802aea7b8b74ed42f/image.png?expires=1772827200&signature=a6868f3776216f4b28076a3d3f74f0b988d78b447c91c7695f6cec66f2b4b991&req=cSkhFMx7moZcFb4f3HP0gCLT5wBZ0ttdialJKOmF5Jb%2Fpsk5r81Gz4vM5R58%0ASzcwJuiSzCD%2BE%2FKsxg%3D%3D%0A)](https://downloads.intercomcdn.com/i/o/596255773/72d4fe3802aea7b8b74ed42f/image.png?expires=1772827200&signature=a6868f3776216f4b28076a3d3f74f0b988d78b447c91c7695f6cec66f2b4b991&req=cSkhFMx7moZcFb4f3HP0gCLT5wBZ0ttdialJKOmF5Jb%2Fpsk5r81Gz4vM5R58%0ASzcwJuiSzCD%2BE%2FKsxg%3D%3D%0A)

The simplest way to add a portal is to press `((` while typing; this will bring up an inline search box which you can use to find the Rem you want to tag the current Rem with.

In other notes systems, features similar to portals may be called _transclusions_ or _mirrors_.

[Learn more about portals](https://help.remnote.com/en/articles/6030742-portals).

# Uploading PDFs to RemNote

Before you can [annotate a PDF](https://help.remnote.com/en/articles/6690975-learning-from-pdfs-and-files-with-the-remnote-reader), you first need to upload it.

RemNote only directly annotates PDFs, websites, and videos, but most other common formats like PowerPoint slides, Word documents, and so on will be automatically converted to PDF when you upload. If you're not sure, try uploading it and see what happens!

You can upload a PDF in several ways:

1.  From the **Create** button in the left sidebar, choose Upload & Annotate File. You can select where the file will be created (by default, in the _Uploaded Files_ folder available from your sidebar).

2.  Within a folder, click the **Upload PDF, PowerPoint, YouTube** button:
    
    [![](https://downloads.intercomcdn.com/i/o/akxf7g7x/1261409220/d218cfb33e3d00fb62a0e376fea5/image.png?expires=1772827200&signature=6b32ebc41d32c37e3585e84a9e46c7826585f13d1b2e108d404e1fff3dafef0f&req=dSIhF81%2BlINdWfMW1HO4zUt0jGV67aa2rOstYKvaLOB8hlWezlTlxmY7kGgp%0Av2y8%0A)](https://downloads.intercomcdn.com/i/o/akxf7g7x/1261409220/d218cfb33e3d00fb62a0e376fea5/image.png?expires=1772827200&signature=6b32ebc41d32c37e3585e84a9e46c7826585f13d1b2e108d404e1fff3dafef0f&req=dSIhF81%2BlINdWfMW1HO4zUt0jGV67aa2rOstYKvaLOB8hlWezlTlxmY7kGgp%0Av2y8%0A)
    
    The file will be created within this folder.  
    ​
    
3.  Within an existing document, click the **Upload** button under the title (you'll need to hover your mouse near the title to see it).
    
    [![](https://downloads.intercomcdn.com/i/o/akxf7g7x/1261409872/7a4c176b1c70bfb27f6421959ce5/image.png?expires=1772827200&signature=f2cc42bd0381813932040f8791953c65e9306cc3eb138353089f1897b4d08568&req=dSIhF81%2BlIlYW%2FMW1HO4ze%2FMKXqoXexwDcxXaDdqwZkLo2FfCdE3wA4Pd4y1%0A4D6I%0A)](https://downloads.intercomcdn.com/i/o/akxf7g7x/1261409872/7a4c176b1c70bfb27f6421959ce5/image.png?expires=1772827200&signature=f2cc42bd0381813932040f8791953c65e9306cc3eb138353089f1897b4d08568&req=dSIhF81%2BlIlYW%2FMW1HO4ze%2FMKXqoXexwDcxXaDdqwZkLo2FfCdE3wA4Pd4y1%0A4D6I%0A)
    
      
    If you haven't typed anything in the document yet, the file you upload and its notes section will replace the current document. Otherwise, it will be created at the [top level](https://help.remnote.com/en/articles/8196578-outlines-and-terminology#h_47e7cb5961) and **linked** to this document (you'll click the PDF button here again to open the PDF you upload).  
    ​
    
4.  **Attach directly to a specific Bullet (Rem)** You are not limited to the document title! You can attach a PDF (or other files) to any specific bullet point in your notes. You can do this using one of the following methods:
    
    -   **Zoom In:** [Zoom into](https://help.remnote.com/en/articles/6030710-zooming-in-to-rems) the specific Rem (click on the bullet point), then click the **Add source** button at the top of the page.
        
    -   **Slash Command:** Simply type `/ipdf` (or `/upload`) at the bullet location and select the file from your computer.
        
        [![](https://downloads.intercomcdn.com/i/o/akxf7g7x/2016080182/ed65aad9d1095ffb6a9015bc0af8/CleanShot+2026-02-01+at+18_02_49%402x.png?expires=1772827200&signature=9f551b9182fd0ef3f3e63c5eb9723b23365f83f08a829bea4eab88c0a9fa5d2d&req=diAmEMl2nYBXW%2FMW1HO4zVX4iR%2Fe286GYDt9TbmsmOVlAK9W6wPFNJp62K7N%0AR51G%0A)](https://downloads.intercomcdn.com/i/o/akxf7g7x/2016080182/ed65aad9d1095ffb6a9015bc0af8/CleanShot+2026-02-01+at+18_02_49%402x.png?expires=1772827200&signature=9f551b9182fd0ef3f3e63c5eb9723b23365f83f08a829bea4eab88c0a9fa5d2d&req=diAmEMl2nYBXW%2FMW1HO4zVX4iR%2Fe286GYDt9TbmsmOVlAK9W6wPFNJp62K7N%0AR51G%0A)
        
    

After choosing any of these options, you can either choose a file on your computer to upload, or paste the URL of a file on the web:

[![](https://downloads.intercomcdn.com/i/o/akxf7g7x/2104918347/8f24fbf7ec8188a143654dfe86f5/CleanShot+2026-02-26+at+00_44_31%402x.png?expires=1772827200&signature=62a37d0d52e9d99f7ab74c7a2e3c086d9c999e83b246d3bd9986d9219f91f2b8&req=diEnEsB%2FlYJbXvMW1HO4zfIpGW9NeGxgKLkbHApuV8fkHDvVsi%2FsG%2FWs1hM0%0AR%2B6cROWb11IMuDJUQ7k%3D%0A)](https://downloads.intercomcdn.com/i/o/akxf7g7x/2104918347/8f24fbf7ec8188a143654dfe86f5/CleanShot+2026-02-26+at+00_44_31%402x.png?expires=1772827200&signature=62a37d0d52e9d99f7ab74c7a2e3c086d9c999e83b246d3bd9986d9219f91f2b8&req=diEnEsB%2FlYJbXvMW1HO4zfIpGW9NeGxgKLkbHApuV8fkHDvVsi%2FsG%2FWs1hM0%0AR%2B6cROWb11IMuDJUQ7k%3D%0A)

In either case, RemNote will grab a copy of the file and store it within your RemNote account so you can access it from any of your devices, then open the file.  
​

The newly uploaded PDF will use the name of the file (on your hard drive, or on the web if you pasted a URL) as its title. If you want to change the name, click on the title in the PDF reader's toolbar to edit it.
